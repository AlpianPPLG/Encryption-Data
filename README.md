
# Data-Encryption Project


Terima kasih telah mengunjungi proyek ini! Aplikasi ini dirancang untuk membantu Anda mengenkripsi dan mendekripsi data menggunakan beberapa algoritma enkripsi populer. Aplikasi ini dibangun dengan menggunakan React JS dan Tailwind CSS, serta memanfaatkan pustaka CryptoJS untuk melakukan operasi enkripsi dan dekripsi.

# Fitur Utama

• Enkripsi Data:

- AES (Advanced Encryption Standard): Algoritma enkripsi simetris yang kuat dan banyak digunakan dalam aplikasi keamanan modern.

- DES (Data Encryption Standard): Algoritma enkripsi simetris yang lebih tua, tetapi masih digunakan dalam beberapa aplikasi legacy.

- Caesar Cipher: Teknik enkripsi sederhana yang menggeser huruf   dalam teks dengan jumlah tertentu, sering digunakan untuk demonstrasi konsep enkripsi dasar.

• Dekripsi Data:

- Mendukung dekripsi data yang telah dienkripsi dengan algoritma AES, DES, dan Caesar Cipher.

• Validasi Input:

- Memastikan bahwa input data tidak kosong sebelum melakukan enkripsi atau dekripsi.

• UI yang Responsif:

- Dibangun menggunakan Tailwind CSS untuk memastikan tampilan yang bersih dan responsif di berbagai perangkat.

• Penjelasan Algoritma:

- Menyediakan penjelasan singkat tentang masing-masing algoritma enkripsi yang digunakan.

• Link ke Akun GitHub:

- Menyertakan tautan ke akun GitHub pengembang di bagian footer aplikasi.

# Cara menggunakan

1. Masukkan Teks: Masukkan teks yang ingin dienkripsi ke dalam kolom input.

2. Enkripsi: Klik tombol "Encrypt" untuk mengenkripsi data. Hasil enkripsi akan ditampilkan dalam tabel di bawahnya.

3. Dekripsi: Untuk mendekripsi data, klik tombol "Decrypt" dan hasil dekripsi akan ditampilkan di tabel terpisah.

4. Validasi: Pastikan kolom input tidak kosong sebelum melakukan enkripsi atau dekripsi.

# Teknologi Yang Digunakan

- React JS: Untuk membangun antarmuka pengguna yang dinamis.
- Tailwind CSS: Untuk desain UI yang responsif dan modern.
- CryptoJS: Untuk operasi enkripsi dan dekripsi.

# Instalasi

Untuk menjalankan proyek ini di lokal Anda, ikuti langkah-langkah berikut:

1. Clone repositori ini:

        git clone https://github.com/username/Encryption-Data.git

2. Pindah ke direktori proyek:
            
        cd repo-name

3. Install dependencies:

        npm install

4. Jalankan aplikasi:

        npm run dev
